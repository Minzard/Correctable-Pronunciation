### 1. 제작 목적
------------------------------
많은 언어 환자들 중 마비말환자는 70%정도를 차지하며 매 환자는 언어 치료를 위해 비용, 접근의 측면에서 어려움을 겪고 있다.  
따라서 손쉬운 언어 치료 접근성, 딥러닝 기술을 사용하여 각 환자에게 알맞는 연속적 데이터 학습을 통한 
환자 개인에게 맞는 솔루션 제공을 하고자 한다. 이를 바탕으로 환자 개인의 언어 치료 부자제로서의  어플리케이션을 개발하고자 한다.
